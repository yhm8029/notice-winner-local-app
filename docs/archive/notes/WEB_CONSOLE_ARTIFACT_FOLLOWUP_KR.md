# 웹 콘솔 산출물 후속 작업 메모

- 문서 역할: 아카이브 노트
- 정본 여부: `archive`
- 이 문서가 답하는 질문: 웹 콘솔 산출물 표시/다운로드 후속 보완 포인트는 무엇이었는가
- 상위 기준 문서: [../../00_CANONICAL_INDEX_KR.md](../../00_CANONICAL_INDEX_KR.md)
- 충돌 시 우선 문서: [../../00_CANONICAL_INDEX_KR.md](../../00_CANONICAL_INDEX_KR.md)

## 배경

- 현재 웹 콘솔은 `GET /api/runs/{run_id}/artifacts` 응답이 있어야 산출물 목록을 표시한다.
- 기본 저장소가 `in_memory`이면 서버 재시작 후 `run_artifacts` 메타데이터가 사라져 화면에서 산출물이 비어 보일 수 있다.
- 실제 파일이 `output/artifacts/` 아래에 남아 있어도, 메타데이터가 없으면 목록과 다운로드 링크가 복원되지 않는다.

## 저장해둘 요구사항

### 1. 산출물은 화면에서 바로 보여야 한다

- 운영자가 선택한 실행의 산출물 목록을 즉시 확인할 수 있어야 한다.
- `project_tracker`이면 `winner_csv`가 보여야 한다.
- `tracker_export`이면 `tracking_excel`이 보여야 한다.
- 가능하면 부모/자식 실행 관계도 같이 보여서 어떤 산출물이 어느 실행에 속하는지 헷갈리지 않게 해야 한다.

### 2. 산출물은 다운로드가 가능해야 한다

- 산출물 카드마다 다운로드 링크 또는 버튼이 보여야 한다.
- 다운로드는 API의 `/api/artifacts/{artifact_id}/download` 경로로 안정적으로 연결되어야 한다.
- 서버 재시작 후에도 다운로드가 가능한 상태를 유지해야 한다.

## 필요한 보완 작업

### A. 백엔드 메타데이터 영속화 또는 복원

- `run_artifacts` 메타데이터가 서버 재시작 후에도 유지되어야 한다.
- 우선순위는 아래 둘 중 하나다.
  - `supabase/postgres` 저장소를 기본 운영 저장소로 사용
  - 로컬 단독 실행용으로 `output/artifacts/` 기준 산출물 메타데이터 복원 로직 추가
- 현재 구현 상태:
  - `SUPABASE_URL`과 `SUPABASE_SECRET_KEY`가 있으면 저장소 기본값이 자동으로 `supabase`로 올라간다.
  - 즉 운영 환경에서는 별도 repository backend env를 쓰지 않아도 메타데이터 영속화 경로를 기본으로 탄다.
  - 로컬 단독 `in_memory` 재시작 복원은 아직 미구현이다.

### B. 프런트 산출물 패널 강화

- 산출물 종류, 파일명, 행 수, MIME 타입을 한글 라벨로 표시
- 부모 실행 산출물과 자식 실행 산출물을 구분해서 표시
- 선택된 실행에 산출물이 없을 때는 "정말 없음"인지, "서버 재시작으로 메타데이터가 사라짐"인지 구분 가능한 안내 문구 제공
- 현재 구현 상태:
  - 선택 실행 / 부모 실행 / 자식 실행 section으로 나눠서 표시한다.
  - empty state에서 현재 artifact backend와 메타데이터 영속 여부를 같이 안내한다.

### C. 화면 내 미리보기

- `winner_csv`: 표 형태 미리보기 제공
- `tracking_excel`: 요약 정보 또는 파싱된 행 미리보기 제공
- 미리보기는 전체 파일 대체가 아니라 운영 확인용 요약 뷰로 제한
- 현재 구현 상태:
  - CSV artifact와 `tracking_excel`은 `/api/artifacts/{artifact_id}/preview` 기반 요약 미리보기를 제공한다.

## 구현 우선순위

1. 산출물 메타데이터 재시작 보존
2. 산출물 목록과 다운로드 안정화
3. CSV/XLSX 화면 미리보기 추가

## 완료 판단 기준

- 웹 콘솔에서 실행 선택 시 산출물 목록이 보인다.
- 각 산출물을 클릭하거나 버튼으로 다운로드할 수 있다.
- 서버를 재시작해도 이전 실행의 산출물 목록과 다운로드가 유지된다.
- 운영자가 터미널 없이 화면에서 산출물 존재 여부를 확인할 수 있다.
