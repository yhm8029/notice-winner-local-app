## 5. 영업 카드 편집 가능성 매트릭스

| 위치 | 현재 사용자 = owner | 현재 사용자 ≠ owner | 관리자 |
| --- | --- | --- | --- |
| 내가 진행 중인 영업 | 메모 수정 가능 | 해당 없음 | 관리자로 보더라도 일반적으론 읽기/강제 조치 |
| 회사 전체 진행 중인 영업 | 수정 불가 | 수정 불가 | 강제 해제/관리 조치만 |
| 전체 영업 대상 프로젝트 | `영업`만 가능 | `영업` 불가 또는 숨김 | 관리 목적 아닌 한 `영업` 직접 수행 안 함 |

## 7. 계약 완료/영업 종료 버튼 매트릭스

| 조건 | 계약 완료 | 영업 종료 | 계약금액 모달 |
| --- | --- | --- | --- |
| owner | 표시 | 표시 | 계약 완료 시 표시 |
| 다른 사용자 | 숨김 | 숨김 | 숨김 |
| 관리자 회사 진행 카드 | 숨김 또는 강제 해제만 | 숨김 | 숨김 |
| 이미 종료/완료 | 숨김 | 숨김 | 숨김 |

추가 규칙:

1. 계약 완료는 계약금액 입력 없이는 저장되지 않는다.
2. 영업 종료는 종료 즉시 상태 변경된다.
3. 종료/완료 후에는 진행 중 영역에서 사라져야 한다.

## 10. 자동 새로고침/동기화 매트릭스

| 영역 | 자동 갱신 | 수동 갱신 | 이벤트 기반 갱신 |
| --- | --- | --- | --- |
| 실행 상세 진행률 | 가능 | 가능 | run 상태 변화 |
| 관리자 영업 집계 | 기본 polling 지양 | 가능 | claim/transfer/close 발생 시 |
| 내가 진행 중인 영업 | 기본 polling 지양 | 가능 | 메모 저장/이관/종료 시 |
| 회사 전체 진행 중인 영업 | 기본 polling 지양 | 가능 | 메모 저장/이관/종료 시 |
| 전체 영업 대상 프로젝트 | 필요 시 수동 | 가능 | claim/close/release 이후 |

## 11. 빈 상태 매트릭스

| 영역 | 빈 상태 문구 방향 |
| --- | --- |
| 내가 진행 중인 영업 | 현재 내가 진행 중인 영업 프로젝트가 없습니다 |
| 회사 전체 진행 중인 영업 | 현재 회사 전체가 진행 중인 영업 프로젝트가 없습니다 |
| 전체 영업 대상 프로젝트 | 조건에 맞는 신규 영업 대상이 없습니다 |
| 종료/완료 정리 | 해당 연도/월의 종료 또는 완료 항목이 없습니다 |
| 사용자 목록 | 등록된 사용자가 없습니다 |
| 초대 목록 | 대기 중인 초대가 없습니다 |
